Bem-vindo(a) ao Thunderbird

Porque precisamos de donativos

O Thunderbird não é mais financiado pela Mozilla. Felizmente existe uma comunidade ativa que o está a melhorar e a mantê-lo a funcionar. Mas, para poder sobreviver a longo prazo, o projeto precisa de ser financiado.

O financiamento do Thunderbird é utilizado para a infraestrutura e equipa que têm de ser suportados para fornecer a aplicação às nossas dezenas de milhões de utilizadores. Para o Thunderbird sobreviver e continuar a evoluir, precisamos do seu apoio e estamos a pedir um donativo seu. Todo o dinheiro que doar irá financiar diretamente o desenvolvimento e infraestrutura do Thunderbird.

Fazer um donativo para o Thunderbird

O Thunderbird é solução de código aberto líder para cliente de e-mail e calendário multi-plataforma, gratuita para utilização empresarial e pessoal. Queremos que se mantenha seguro e que continue a melhorar ainda mais. Um donativo irá permitir a contratação de programadores, o pagamento da infraestrutura e continuar a melhorar.

O Thunderbird é financiado por utilizadores como você! Se gosta do Thunderbird, considere fazer um donativo. A melhor forma para si de garantir que o Thunderbird se mantém disponível é fazendo um donativo.

Participe

O Thunderbird é um projeto de código aberto, o que significa que qualquer pessoa pode contribuir com ideias, designs, código e tempo para ajudar outros utilizadores. Faça parte de uma equipa que apoia os utilizadores do Thunderbird e que tornam o Thunderbird fantástico. Junte-se a nós

Obter apoio

Precisa de ajuda em alguma coisa com o Thunderbird? Nós temos uma página de apoio onde pode colocar questões e ler artigos de apoio para saber como utilizar o Thunderbird.

Reportar um problema

Acha que encontrou um problema com o Thunderbird? Crie um relatório de falha para que possamos garantir que é resolvido. Ajude-nos a identificar e a corrigir erros para tornar o Thunderbird melhor para todos.
