Bienvenido a Thunderbird

Por qué necesitamos donaciones

Thunderbird ya no es financiado por Mozilla. Afortunadamente existe una comunidad activa que lo mantiene y sigue desarrollándolo. Pero para sobrevivir a largo plazo, el proyecto necesita financiamiento.

El financiamiento de Thunderbird va directamente a mantener la infraestructura y el personal para que la aplicación sea entregada a decenas de millones de usuarios. Para que Thunderbird sobreviva y siga evolucionando, necesitamos tu apoyo y te pedimos una donación. Todo el dinero donado irá directamente a financiar el desarrollo y la infraestructura para Thunderbird.

Dona a Thunderbird

Thunderbird es el líder en clientes de correo y calendario multiplataforma y de código abierto, gratuito para uso personal y empresarial. Queremos que se mantenga seguro y sea aún mejor. Una donación nos permitirá contratar desarrolladores, pagar por la infraestructura y continuar mejorando.

¡Thunderbird es financiado por usuarios como tu! Si te gusta Thunderbird, por favor considera realizar una donación. La mejor manera de asegurar que Thunderbird permanece disponible es hacer una donación.

Participar

Thunderbird es un proyecto de código abierto, lo que significa que todos pueden contribuir ideas, diseños, código u tiempo para ayudar a otros usuarios. Se parte de un equipo que apoya a los usuarios de Thunderbird y lo hace asombroso. Únete

Recibe ayuda

¿Necesitas ayuda con algo en Thunderbird? Tenemos una página de soporte donde puedes preguntar y leer artículos para aprender a usar Thunderbird.

Reportar un error

¿Crees que encontraste un problema con Thunderbird? Crea un reporte de error para que podamos asegurarnos de que sea resuelto. Ayudándonos a identificar y solucionar errores mejoramos Thunderbird para todos.
