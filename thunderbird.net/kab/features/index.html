Almad fessus

Amarag n twila n umiḍan imayl

Send ad d-ban tmahilt-agi, yessefk ad tissineḍ iɣewwaṛen IMAP, SMTP, SSL/TLS n imḍanen inek. Tura mudd kan isem-ik, tansa yinek n yimayl akked wawal uffir, sakin amarag amaynut ad yexdem ayen nniden s unadi di taffa n isefka n iɣewwaṛen n usaǧǧaw inek n yimayl.

Imedlis n tenṣa ɣef afus

Imedlis n tensa ɣef afus d allal arurad, fessusen akken ad ternud imdanen ar imedlis n tensa. Rnu anermis s usiti ɣef uqerru n yizen i d-remseḍ. Ma tsiteḍ snat n tikal tzemreḍ ad ternud ugar n telɣut am tewlaft, azemz n tlalit, d wayen nniḍen.

Asmekti n imeddayen

Anagraw n ulɣu ad yesleḍ akken ad inadi awalen am amedday, isemli, afaylu (neɣ ismawen n isiɣzaf n ifuyla) udiɣ ak-id-immekti ad tsedduḍ isemli ma yella tettuḍ.

Iccaren d unadi

Iccaren

Ma tḥemleḍ tunigin s yiccaren n Firefox, ad tiḥmileḍ iccaren n Thunderbird. Asnas n yeznan s yiccaren n Thunderbird ak-yeǧǧ ad d-saliḍ iznan inek yal yiwen deg iccer akken ad tizmireḍ ad tedduḍ seg yiwen ar wayeḍ s zreb. Aqlak tettarraḍ i yizen daɣen tesriḍ ad twaliḍ izen nniḍen aqbuṛ? Asnas n yeznan s yiccaren ak-yeǧǧ ad tḥ€rzeḍ deqs n yiccaren ldin deg iwen n usfaylu akken ad yifsus useqdec.

Ticki tsiteḍ snat n tikal ɣef yizen neɣ tsendeḍ ɣef tqeffalt Kcem ticki izen yettwafren, izen ad d-yeldi deg iccer amaynut. Yiwen n usiti ayeffus ɣef yeznan neɣ ikaramen ad -yeldi deg iccer deg ugilal.

Ticki tmedleḍ Thunderbird, iccaren d-ibanen ad ttwakelsen sakin ad d-uɣalen di tnekra n Thunderbird. Yella wumuɣ amaynut Iccer deg ufeggag n yiccaren akken ak-d-imudd afus ticki tettedduḍ seg iccer ara wayeḍ.

Afeggag n ifecka n imzizdig arurad

Afeggag n ifecka n imzizgid arurad ak-yeǧǧ ad tezizdgeḍ iznan inek s zreb. Sekcem di tazwara awalen deg ufeggag n unadiarurad sakin igmaḍ ad d-banen imiren kan. Tzemreḍ daɣen ad tezizdgeḍ s Iznan imaynuten, Tibzimin, akked yemdanen n imedlis inek n tensa. Tzemreḍ daɣen ad sekelseḍ imzizdig akked ad tesqedceḍ tikelt nniḍen.

Nadi

Agrudem n unadi n Thunderbird yegber ifecka n uzizdeg akked usmizzwer askudan akked ad d-yaf izen tettnadiḍ. Thunderbird irennu amatar i yeznan inek akken ak-d-imudd afus deg unadi n s zreb. Igmaḍ inek n unadi ad d-ttwaseknen deg iccer amaynut akken ad tizmireḍ ad ten-twaliḍ udiɣ ad d-uɣaleḍ s zreb ar igmaḍ n unadi.

Aḥraz n yizen

Ma tɣileḍ tesriḍ aḥraz n yizen ur tebɣiḍ ara ad yili di tebwaṭ inek n urmes war ma tekkseṭ, ḥerz-it! Aḥraz ak-dimudd afus deg usefrek n tebwaṭ inek n yeznan udiɣ ad yawi izen ar unagraw amaynut n ikaramen n uḥraz.

Akken ad tḥerzeḍ izen, Fren taqeffalt ḥrez neɣ tekki ɣef tqeffalt ‘A’ .

Amsefrak n urmud

Amsefrak n urmud ad isekles tamyigawt gar Thunderbird akked usaǧǧaw inek n yimayl deg yiwen n umḍiq. Dayen, ur tesrid ara ad tɛasseḍ iznan inek. Yiwen kan n umḍiq anda ara twaliḍ.

Talɣa n Thunderbird

Amsefrak n izegrar

Taɣellist d ummesten n yimayl inek

Tabaḍnit iǧehden

Thunderbird yetttḥărab ɣef tudert inek tusligt akked uselkim inek mgal askan n tugniwin ur ilaqemn ara. Akken ad neḍmen ammesten-agi, Thunderbird isewḥal s wudem awurman tugniwin tinmeggagin yellan deg iznan.

Ammesten mgal tandit

Thunderbird ak-immesten mgal iskamen (iznan n tandit) ara k-yeǧǧen ad tamneḍ am akken izen yekka-d seg wid yettwamanen akken ad awin talɣut tudmawant neɣ tabaḍnit inek neɣ ad akkren tamagit inek. Iznan-agi ttwammalen d ixeṣṣaren di Thunderbird. Thunderbird yeqqaṛ ɣur-wat ticki aseɣwen URL yellan deg izen yettawi ar usmel nniḍen yettwammlen deg izen.

Aleqqem awurman

Anagraw n uleqqem n Thunderbird ad isenqed ma yella tseqdaceḍ lqem aneggaru iheggan udiɣ ak-id-yelɣu s wudem awurman ticki lqem n tɣellist yella. Teɣzi n ileqman-agi n tɣellist meẓẓi aṭas (di tanumi gar 200 KAṬ akked 700 KAṬ). Ileqman-agi gebren ayen ilaqen kan akken ad yerr lqem d arurad i usider akked usebded. Anagraw n uleqqem awurman yettmuddu-d ileqman i Thunderbird di Windows, macOS akked Linux deg ugar n 40 n tutlayin.

Ḍegger ispamen

Ifecka n tuṭṭfa n yeznan ispamen di Thunderbird ttwaleqmen akken ad izmiren ad ḥarben mgal ispamen. Yal izen id-yettwaremsen ad iɛeddi ɣef imzizdig amgal-aspam n Thunderbird. Yal tikelt i tceṛḍeḍ izen amzun d aspam, Thunderbird "ad yissin" sakin ad isnerni azizdeg ines, ayen ara kyeǧǧen ad teɣreḍ iznan inek ilaqen. Thunderbird yezmer daɣen ad iseqdec imzizdigen amgal-ispamen n usaǧǧaw inek n yimayl akken ur tettezziḍ ara ad d-remseḍ ispamen di tebwaṭ inek n urmas.

Aɣbalu yeldin

Thunderbird yekka-d si tneflit n useɣẓan ilelli i ggan wagimen n isneflayen ifazen iḥemlen aseɣẓan, d imazzagen di tɣellist i d-yekkan seg umaḍal meṛṛa. Tamezdagnut nneɣ n imazzagend turmiṭ udiɣ teldi, tettmuddu-yaɣ-d afus akken ad neḍmen taɣlellist akked uleqqem n ifarisen-nneɣ s zreb, ayagi daɣen s ufaṛes n tegnitt n ifecka ifazen n usenqed akked usekyed n t qellist akken ad neseǧhed taɣellist tamatut.
